# Postfix Notation
Postfix Notation (Reverse Polish Notation) Calculator.

## :clipboard: Requirements 
- Rust

## :rocket: Usage
Run the program using the following command.
```sh
cargo run
```

The expression `2+2` can be expressed like this:
```
2 2 +
```

Let's take a look at `pow(2, 5)`:
```
2 2 * 2 * 2 * 2 *
```
